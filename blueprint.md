# Project Blueprint

## Overview

사용자의 개인 정보를 바탕으로 최상의 화장품을 추천해주는 웹 애플리케이션입니다.

## Implemented Features

*   **Initial project setup:** `index.html`, `style.css`, `main.js` 기반 구축.
*   **맞춤형 화장품 추천 기능:** 
    - 사용자가 피부 고민(건조, 여드름, 미백 등)을 입력하면 관련 키워드를 분석.
    - 고가, 중가, 저가 총 3가지 가격대의 제품을 선별하여 화면에 표시.
    - 반응형 제품 카드 레이아웃 적용.
